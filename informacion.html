Información Técnica del Portafolio

Detalles Técnicos del Sitio

Elemento Descripción
Tecnologías utilizadas El sitio fue desarrollado utilizando HTML5 para la estructura, CSS3 para el diseño visual, JavaScript para la interactividad y el framework Bootstrap para lograr un diseño responsivo y adaptable a distintos dispositivos.
Plataforma de publicación El portafolio se encuentra alojado en GitHub Pages, lo que permite que esté disponible en línea mediante una URL pública como evidencia digital del trabajo académico.
Control de versiones Se utilizó Git para el control de versiones del proyecto y GitHub como repositorio remoto, permitiendo el seguimiento de cambios, respaldo del código y organización del desarrollo del sitio.
Seguridad del sitio (HTTPS) El sitio utiliza el protocolo HTTPS, habilitado automáticamente por GitHub Pages mediante un certificado SSL. Esto garantiza que la información transmitida entre el usuario y el sitio se encuentre cifrada y protegida contra posibles interceptaciones.
Importancia de la seguridad La implementación de HTTPS forma parte de las buenas prácticas de seguridad informática, asegurando la integridad y confidencialidad de la información, especialmente relevante en un portafolio relacionado con ciberseguridad.
